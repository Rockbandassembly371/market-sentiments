# AION-Sentiment-IN-v2

Financial sentiment analysis model tuned on Indian market news.

## Model Details

**Developer:** AION Analytics Open-Source Contributors

**Model Type:** Transformer-based sequence classification (DistilBERT)

**Input:** Text (financial news headlines)

**Output:** Sentiment classification (negative, neutral, positive)

**License:** Apache License 2.0

## Model Description

AION-Sentiment-IN-v2 is a sentiment analysis model specifically trained on Indian financial news headlines. The model was retrained in March 2026 after discovering labeling issues in the original training data.

### Key Features

- **India-Tuned:** Trained on 1M+ Indian financial news headlines from 2024-2026
- **VADER-Labeled:** Training labels generated using VADER sentiment lexicon (not dependent on potentially biased human labels)
- **Three-Class Classification:** negative, neutral, positive
- **98% Accuracy:** Validated on 20% holdout test set

### Training Data

- **Source:** ClickHouse table `aion_master.news_master_v1` (Indian financial news corpus)
- **Size:** 1,029,611 headlines (823,688 train / 205,923 validation)
- **Labeling Method:** VADER sentiment lexicon with thresholds:
  - compound ≥ 0.05 → positive
  - compound ≤ -0.05 → negative
  - otherwise → neutral
- **Time Period:** 2024-01-01 to 2026-03-31
- **Label Distribution:**
  - negative: 139,385 (13.5%)
  - neutral: 305,131 (29.7%)
  - positive: 379,172 (36.8%)

### Training Procedure

- **Framework:** PyTorch with HuggingFace Transformers
- **Base Architecture:** DistilBERT (6-layer transformer)
- **Epochs:** 3
- **Batch Size:** 32
- **Learning Rate:** 2e-05
- **Hardware:** Apple M4 (MPS acceleration)
- **Training Time:** ~10 hours

## Limitations

1. **Overconfidence:** Model often returns 100% confidence scores due to softmax saturation. Use confidence scores as relative indicators rather than absolute probabilities.

2. **Domain Specificity:** Works best on Indian financial news. Performance may degrade on:
   - Non-financial text
   - Non-Indian market context
   - Social media or informal text

3. **Nuanced Headlines:** May misclassify headlines with:
   - Mixed sentiment (positive + negative elements)
   - Sarcasm or irony
   - Cultural references specific to Indian markets

4. **Training Data Bias:** Labels generated by VADER lexicon, which has known limitations:
   - May not capture financial context nuances
   - Compound score thresholds may not generalize

## Use Cases

### Intended Uses

1. **Financial News Analysis:** Automated sentiment classification of Indian financial news headlines
2. **Market Research:** Sentiment trend analysis across sectors or time periods
3. **Portfolio Monitoring:** Track sentiment around specific stocks or sectors
4. **Academic Research:** Study sentiment-market relationships in emerging markets

### Out-of-Scope Uses

- Direct trading decisions without human oversight
- Non-financial sentiment analysis
- Non-English text
- Real-time trading systems (use internal production systems)

## Usage

### Using aion-sentiment Package (Recommended)

```python
from aion_sentiment import SentimentAnalyzer

analyzer = SentimentAnalyzer()
result = analyzer.predict("RBI hikes repo rate by 25 bps")
print(result)
# {'label': 'negative', 'confidence': 0.89}
```

### Using Transformers Directly

```python
from transformers import pipeline

classifier = pipeline(
    "text-classification",
    model="aion-analytics/aion-sentiment-in-v2"
)
result = classifier("RBI hikes repo rate by 25 bps")
print(result)
# [{'label': 'LABEL_0', 'score': 0.89}]
# Note: LABEL_0=negative, LABEL_1=neutral, LABEL_2=positive
```

### Batch Processing

```python
from aion_sentiment import SentimentAnalyzer
import pandas as pd

analyzer = SentimentAnalyzer()

# Single prediction
result = analyzer.predict("Stock market reaches all-time high")

# Batch prediction
texts = [
    "RBI hikes repo rate",
    "Stock market crashes",
    "TCS reports record earnings"
]
results = analyzer.predict(texts)

# DataFrame integration
df = pd.DataFrame({'headline': texts})
df = analyzer.analyze(df, text_column='headline')
```

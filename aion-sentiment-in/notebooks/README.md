# AION Notebooks

This directory contains Jupyter notebooks for exploring, analyzing, and developing
models for the AION Sentiment Analysis project.

## Notebook Organization

```
notebooks/
├── 01_data_exploration.ipynb     # Exploratory data analysis
├── 02_model_training.ipynb       # Model training experiments
├── 03_model_evaluation.ipynb     # Model evaluation and metrics
└── README.md                     # This file
```

## Getting Started

1. Install notebook dependencies:
   ```bash
   pip install -r requirements-notebooks.txt
   ```

2. Start Jupyter:
   ```bash
   jupyter notebook
   ```

3. Navigate through notebooks in order for a complete workflow

## Notebook Guidelines

- Follow AION coding standards
- Include markdown cells explaining each step
- Document all assumptions and parameters
- Save outputs for reproducibility

## Contributing

When adding new notebooks:
1. Use descriptive names with numbered prefixes
2. Include a clear title and description
3. Document dependencies and requirements
4. Test execution from start to finish

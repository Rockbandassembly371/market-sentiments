# AION Data Directory

This directory contains data files for the AION Sentiment Analysis project.

## Directory Structure

```
data/
├── README.md                 # This file
├── raw/                      # Raw extracted data (not committed to git)
│   └── raw_extracted.csv     # Raw data from database extraction
├── processed/                # Processed datasets (not committed to git)
│   ├── train.csv             # Training dataset
│   └── val.csv               # Validation dataset
└── lexicons/                 # Emotion lexicons
    └── NRC-Emotion-Lexicon.txt  # NRC Emotion Lexicon (downloaded)
```

## Data Files

### Raw Data (`raw/`)

Raw data extracted from the `aion_analytics.news_sentiment` database.

**Source**: Execute `extract_data.sql` against your database

**Schema**:
| Column | Type | Description |
|--------|------|-------------|
| headline | string | News article headline |
| publish_date | date | Publication date |
| sentiment_label | string | Raw sentiment label |
| confidence_score | float | Model confidence (0.0-1.0) |
| ticker | string | Stock ticker symbol |
| close_price | float | Stock closing price |
| returns_1d | float | 1-day return (%) |
| returns_3d | float | 3-day return (%) |
| returns_5d | float | 5-day return (%) |

### Processed Data (`processed/`)

Processed datasets ready for model training.

**Generated by**: `python prepare_data.py --input data/raw/raw_extracted.csv --output data/processed/`

**Schema** (train.csv, val.csv):
| Column | Type | Description |
|--------|------|-------------|
| headline | string | Original news headline |
| headline_clean | string | Preprocessed headline |
| label | int | Encoded sentiment (0=negative, 1=neutral, 2=positive) |
| publish_date | date | Publication date |
| ticker | string | Stock ticker symbol |
| ... | ... | Additional columns from raw data |

### Lexicons (`lexicons/`)

Emotion lexicons for fine-grained emotion analysis.

#### NRC Emotion Lexicon

**File**: `NRC-Emotion-Lexicon.txt`

**Description**: Word-emotion association lexicon with eight emotion categories.

**Download**: 
- Automatic: The `emotion_utils.py` script will attempt to download automatically
- Manual: Visit https://saifmohammad.com/WebPages/NRC-Emotion-Lexicon.htm

**Emotion Categories**:
1. **anger** - Feelings of hostility, irritation, or rage
2. **fear** - Feelings of anxiety, worry, or apprehension
3. **joy** - Feelings of happiness, delight, or pleasure
4. **sadness** - Feelings of sorrow, grief, or unhappiness
5. **trust** - Feelings of acceptance, confidence, or reliance
6. **disgust** - Feelings of revulsion, contempt, or aversion
7. **surprise** - Feelings of astonishment or unexpectedness
8. **anticipation** - Feelings of expectation or looking forward

**Format**: Tab-separated values
```
word    emotion    value
abandon anger      1
abandon fear       1
abandon sadness    1
...
```

## Git Ignore

The following files are ignored by git (see `.gitignore`):
- `data/raw/*` - Raw data (may contain sensitive information)
- `data/processed/*` - Generated datasets (can be regenerated)
- `data/lexicons/*` - Downloaded lexicons (can be re-downloaded)

## Data Preparation Workflow

1. **Extract raw data**:
   ```bash
   psql -h localhost -U username -d aion_analytics -f extract_data.sql
   # Save output to data/raw/raw_extracted.csv
   ```

2. **Prepare datasets**:
   ```bash
   python prepare_data.py --input data/raw/raw_extracted.csv --output data/processed/
   ```

3. **Verify data**:
   ```bash
   python -c "import pandas as pd; print(pd.read_csv('data/processed/train.csv').head())"
   ```
